Algorithm (Detailed Design)

Module Ref 0 - AverageSystem

Begin
Call GetNumber1 Using Number1
Call GetNumber2 Using Number2
Call CalcAverage Using Number1, Number2 and Average
Call DisplayAverage Using Average
End

Module Ref 1 - GetNumber1

Begin
Display 'Please enter first number:'
Get Number1
End

Module Ref 2 - GetNumber2

Begin
Display 'Please enter second number:'
Get Number2
End

Module Ref 3 - CalcAverage

Begin
Calc Average as Number1 + Number2 / 2
End

Module Ref 4 - DisplayAverage

Begin
Display 'The average is ', Average
End
