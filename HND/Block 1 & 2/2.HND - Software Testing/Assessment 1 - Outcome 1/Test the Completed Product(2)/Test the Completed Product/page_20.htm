Desk Checking

Desk checking is a paper and pencil exercise where the programmer works through the program by hand keeping track of the values of each variable and the statements that are executed.

Each line to be tested is numbered and a record kept of the work through, eg:

 
Program NumberCountingSystem ;

Var
Number, TotalPositive, TotalNegative : Integer ;

Begin { NumberCountingSystem }

01. TotalPositive := 0 ;
02. TotalNegative := 0 ;
03. Write ('Enter Number (Type 0 To Terminate) ') ;
04. Readln (Number) ;

05. While (Number &#60;&#62; 0) Do

06. Begin { While }

07. If (Number > 0) Then
08. TotalPositive := TotalPositive + 1
09. Else
10. TotalNegative := TotalNegative + 1 ;

11. End; { While }

12. Writeln ('Total Positive Integers = ', TotalPositive:0) ;
13. Writeln ('Total Negative Integers = ', TotalNegative:0) ;

End. { NumberCountingSystem }
