Program Updates

Update 1

The procedure GetTechData must be amended to include:

 
Display 'Enter tech hour code (1 to 6 only): '
Get (TechHourCode)

Update 2

You must add a CalculateJobCost procedure in an appropriate place in your program code.

 
Module CalculateJobCost 
{ TechTable - In/Out Parameter }
{ NoOfRecords - In }

Begin { CalculateJobCost }

for Count is equal to 1 to NoOfRecords do
with TechTable[Count]) do
Calc TotalJobCost as TechHourCode * CostPerHour

Update 3

You must modify the DisplayTechInfo procedure to display the new information: TechHourCode and TotalJobCost: 

 
Display 'Tech hour code: ', TechHourCode
Display 'Total job code: ', TotalJobCost

Update 4

You must modify your menu system to include the extra option for the user to calculate the job cost.

You must incorporate a procedure that will validate user input. Note: This procedure should be used at least once, eg: for your menu of for the data entry for the data entry of the TechHourCode.
